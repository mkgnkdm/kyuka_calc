# 年休計算機 — 年次有給休暇の付与日数計算ツール

入社日と週の働き方を入れるだけで、年次有給休暇の**付与日**と**付与日数**（比例付与を含む）を、
**根拠の法定テーブルつき**で表示するWebツールです。

<!-- 公開時: スクリーンショットをここに1枚 -->

## 特徴 — 「根拠を隠さない計算機」

労務系の計算ツールは「答えの数字」だけを出すものが多いですが、実務では**根拠を確認できること**が
信頼の条件です。本ツールは次の設計をとっています。

1. **法定テーブルをコードに書かない** — 付与日数の表は [`rules.json`](rules.json) で管理し、
   出典（労働基準法39条・施行規則24条の3）と**専門家による確認日**を持ちます。
   法改正時はこのファイルの更新だけで対応できます
2. **根拠を画面に表示する** — 計算結果の下で法定テーブルそのものと出典を確認できます。
   テーブルが確認前（draft）の間は、画面に警告が出て実務利用を止めます
3. **判定しないことを明示する** — 出勤率8割要件・会社ごとの基準日統一方式は本ツールの
   守備範囲外であることを、隠さず画面に書いています

## 対応している計算

- 通常付与（週5日以上または週30時間以上）: 勤続6か月〜6年6か月以上の7段階
- 比例付与（週30時間未満かつ週1〜4日）: 4区分×7段階
- 入社日起算の法定原則による付与日（月末入社の丸め・うるう年対応）
- 基準日方式（一斉付与）: 初回のみ法定どおり、2回目以降を会社の基準日に統一した場合の
  付与日・付与日数（法定日からの前倒しを表示。斉一的取扱いが適法な「前倒しのみ」であることを
  プロパティテストで担保）

## 対応していないこと（意図的なスコープ外）

- 出勤率8割以上の要件判定（画面に注記のみ）
- 入社時から基準日に寄せる方式・複数基準日（年2回）での付与日計算
- 年5日の取得義務の管理

## プライバシー

- 入力されたデータは**一切保存しません**（DBなし・セッションなし）
- 計算はステートレスで、結果はURLで共有できます

## 技術構成

- Python / Flask（`leave.py` = 計算エンジンはFlask非依存の純ロジック）
- テスト: pytest 71件（法定表の構造検証・境界ケース〔月末入社・うるう年・勤続上限〕・
  基準日方式の不変条件〔前倒しのみで法定を上回らないこと〕・画面の歩き通しシナリオ検証）
- 実行: `pip install -r requirements.txt` → `python app.py` → http://127.0.0.1:5011

## 免責

本ツールは法定の原則による目安を表示するものです。実際の付与は就業規則・出勤率・基準日方式の
採用などにより異なる場合があります。正確な日数は就業規則をご確認のうえ、必要に応じて
社会保険労務士等の専門家にご相談ください。本ツールの利用により生じたいかなる結果についても、
作者は責任を負いません。
